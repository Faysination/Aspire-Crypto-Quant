# Binance Crypto Auto Bot

A fully automated, regime-adaptive quantitative trading bot for Binance Spot and Futures, featuring a beautiful full-width local dashboard to monitor trades and portfolio performance.

## Setup

1. Run `start.bat`
2. Open `http://127.0.0.1:5001` in your browser.
